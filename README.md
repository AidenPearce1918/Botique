# Botique - Exclusive Ladies Fashion

A modern, elegant website for Mumbai's premier ladies' fashion boutique specializing in exquisite Indian attire.

## ✨ Features

- **Exclusive Ladies' Collections**: Curated selection of sarees, kurtis, lehengas, and accessories
- **Interactive Filtering**: Filter products by category (Sarees, Kurtis, Lehengas, Accessories)
- **Modern Design**: Elegant UI with premium color palette and smooth animations
- **Responsive Layout**: Optimized for all devices
- **Product Modal**: Detailed product view with inquiry functionality
- **Smooth Scrolling**: Enhanced navigation experience

## 🎨 Design Highlights

- Premium color scheme with gold and brown accents
- Elegant typography using Playfair Display and Poppins fonts
- Subtle animations and hover effects
- Card-based product layout with category tags
- Gradient backgrounds and shadow effects

## 📁 Files

- `index.html`: Main HTML structure with modern semantic markup
- `style.css`: Modern CSS with custom properties, animations, and responsive design
- `script.js`: Interactive JavaScript for filtering, modals, and smooth scrolling

## 🚀 How to View

1. Open `index.html` in a modern web browser
2. Or run a local server: `python3 -m http.server 8000`
3. Visit `http://localhost:8000`

- **Kurtis**: Embroidered and printed kurti sets
